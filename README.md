# Preparar el entorno

1. Instalar Docker
2. Descargar repo
3. Desde la consola o el power shell, ve a la carpeta donde tienes el proyecto
4. ejecuta docker-compose up -d
5. ejecuta ./init.sh
6. abre la web: http://localhost:8888/listadopalabras.php
